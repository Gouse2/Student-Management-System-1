# Student Management System (Desktop App)

A complete **Java Desktop Application** built with:
- **Swing** – Beautiful GUI
- **Hibernate** – Database handling
- **MySQL** – Backend storage
- **JOptionPane** – Clean input dialogs

## Features
- Add Student
- View Student by ID
- Update Marks & Type
- Delete Student
- View All Students
- Clean & Professional UI

## Technologies Used
- Java SE
- Swing (GUI)
- Hibernate ORM
- MySQL Database
- Eclipse IDE

## How to Run
1. Configure `hibernate.cfg.xml` with your MySQL details
2. Run `App.java` as Java Application
3. Enjoy the GUI!
